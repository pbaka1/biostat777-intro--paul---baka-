## Project Description

This repository contains my personal website built as part of **Biostat 777 – Introduction to Data Science in Public Health**.\
The website highlights my professional background and demonstrates practical data analysis skills using R and Quarto.

The site includes: 1. **Home** – Introduction and personal summary\
2. **About** – Expanded biography, education history, and research interests\
3. **Example Analysis** – Reproducible data analysis using the Palmer Penguins dataset to explore species differences and environmental patterns
